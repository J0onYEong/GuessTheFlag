# GuessTheFlag
Project2 in HackingWithSwift
# 기능
- 먼저 국기의 이름이 주어지고 랜덤으로 주어지는 3가지 국기중 이름과 일치하는 국기를 찾아내는 게임이다.
# 특이사항
- 게임은 총 8번 치뤄지며 8번의 게임을 통해 산출된 점수를 최종 점수로 취급한다.
- alert창을 활용하여 정답유무와 최종 결과화면을 표시하는 방식으로 동작한다.
